# GamesApp

# About the project

https://gamesapp-production.up.railway.app

GamesApp is a back-end application (front-end will be built soon) developed during the **INTENSIVÃƒO JAVA SPRING** course, an event organized by [Devsuperior Java Spring](https://www.youtube.com/@DevsuperiorJavaSpring " DevSuperior youtube channel").
